# IDirectFBGL2 接口

`IDirectFBGL2` 接口提供了对DirectFB中OpenGL ES 2.x渲染上下文的操作和管理。

## 上下文

| 方法              | 描述                     |
| ----------------- | ------------------------ |
| **CreateContext** | 创建一个新的渲染上下文。 |

## 接口

| 方法               | 描述                   |
| ------------------ | ---------------------- |
| **GetProcAddress** | 获取OpenGL函数的地址。 |

## 纹理

| 方法               | 描述                         |
| ------------------ | ---------------------------- |
| **TextureSurface** | 将表面缓冲区绑定到纹理对象。 |
